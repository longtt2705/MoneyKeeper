# MoneyKeeper - Money management app

## :wrench: How to run 

Clone project, then cd to project & run

```
npm start
```

Or an build APK
[APK build](https://drive.google.com/file/d/1TvfemFuJLU1uMIYzvux4XgvcBIKJqTgt/view?usp=sharing)
